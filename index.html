    atlas-vim

    Vim with Setup for Atlas Server of Sheridan College(this is only for sheridan college student) Make of backup of .vim folder and .vimrc(if you have one in your home folder)

    Instruction

    Step 1

    copy the download zip link in the right on github page(in this page) then go to terminal/putty , login to your atlas and run

    wget https://github.com/juliosueiras/atlas-vim/archive/master.zip

    Step 2

    next execute

    unzip master.zip && mv atlas-vim-master atlas-vim

    Step 3

    last go inside of the folder and execute BEWARE this will overwrite .vim and .vimrc in your home folder(if you have one) make sure to backup it first

    ./install.sh
